### bool type

#### 사용 예시
```javascript
true
false
3 > 7
3 < 7
3 == 3
```

```javascript
true
false
false
true
true
```

#### 신기한 것(파이썬과의 다른 점)
```javascript
3 == '3'
3 === '3'
```
```javascript
true
false
```
> `==` 는 값이 같은지 여부만을 판단하는 것이고
> `===`는 값과 데이터 타입이 모두 같은지 여부를 판단하는 것이다.

#### and or not
- and : &&    /   둘다 참이여야 true
- or : ||     /   하나만 참이면 true
- not : !false(true)    /    반대로 뒤집어주는 역할
